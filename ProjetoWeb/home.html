Sistema Banco Nix
Bem vindo ao sistema do Banco Nix. Aqui você terá acesso a todas as suas Transferências dentro do banco.
Todas as Transferência listadas são apenas de seu usuário. Caso deseje acessar as trasferências de outro usuário, por favor faça login com o mesmo.

O botão de Nova Transferência irá te abrir uma tela para inserir todos os dados do Pagador, Beneficiario e o Valor da transferência que deseja realizar.

Para editar algum item, clique sobre o registro desejado na tabela Lista de Transferências para abrir a tela de edição. Será a mesma tela de Nova Transferência, porém com o registro selecionado.

Quando aberta a tela de Novo/Editar Transferência, basta preencher todos os campos e clicar em Salvar para salvar seu registro. Para não o salvar basta clicar em Cancelar ou clicar fora da tela.

A esquerda de cada item consta um ícone de uma Lixeira: É onde poderá deletar seu registro. Clique no mesmo para aparecer uma tela de confirmação de deleção. Clique em Deletar caso queira realmente o deletar.

Abaixo da tabela Nova Transferência há uma linha que mostrará o número de registros que constam para seu usuário e quais estão o sendo exibido. A direita desta linha há dois botões para alterar a página em que se encontra.

Faça bom uso do sistema. Obrigado.
